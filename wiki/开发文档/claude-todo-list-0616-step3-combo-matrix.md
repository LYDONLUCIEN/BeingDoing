## 决策摘要

| # | 决策 | 结论 |
|---|------|------|
| 1 | 组合范围 | 3×5=15个自由组合，热爱优先遍历 |
| 2 | combo_id 格式 | `passion_idx` + `strength_idx` 拼接（如 `"02"`） |
| 3 | 消息存储 | 一个文件，每条消息带 combo_id |
| 4 | 流程分段 | filter_step=3 → filter_step=3b → step4+ |
| 5 | 进度条 | 3和3b统一显示在3，可互相切换 |
| 6 | AI上下文 | 完全隔离，每个组合只看自己的对话+用户背景 |
| 7 | 引导语 | 固定模板，只替换完成数量 n |
| 8 | 结论卡确认 | >=5字可确认，chips 替换填入，textarea 编辑 |
| 9 | 跳过状态 | 保留文字 + 卡片灰显 + "已跳过"标签 |
| 10 | 全部提交 | 始终可点，弹窗提示纯空和有字跳过 |
| 11 | 3b表格列 | 热爱、优势、假设（无匹配性） |
| 12 | 3b→step4 衔接 | 在3b基础上加列，复用现有逻辑 |
| 13 | neg gate | 仅3b触发，3不触发 |
| 14 | 纯空组合 | 进入3b后不显示在表格中 |
| 15 | 3↔3b 切换 | 状态保持不变 |
