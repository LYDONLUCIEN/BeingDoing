# 常见问题 (FAQ)

## 🔧 技术问题

### Q: 后端启动失败，提示端口被占用？
A: 修改端口或关闭占用端口的程序：
```bash
# 修改端口
uvicorn app.main:app --port 8001

# 或查找并关闭占用进程
# Windows:
netstat -ano | findstr :8000
taskkill /PID <进程ID> /F
```

### Q: 前端启动失败，提示模块未找到？
A: 重新安装依赖：
```bash
cd src/frontend
rm -rf node_modules .next
npm install
```

### Q: 数据库连接失败？
A: 检查数据库配置：
- SQLite: 确保 `data/` 目录存在且有写入权限
- PostgreSQL: 检查连接字符串和数据库是否已创建

### Q: LLM API调用失败？
A: 检查：
1. `OPENAI_API_KEY` 是否正确配置
2. API密钥是否有效
3. 网络连接是否正常
4. API额度是否充足

### Q: 对话记录没有保存？
A: 检查：
1. `data/conversations/` 目录是否存在
2. 目录是否有写入权限
3. 查看后端日志是否有错误

## 🎨 功能问题

### Q: 智能助手没有回复？
A: 可能原因：
1. LLM API调用失败（检查API密钥）
2. 网络问题
3. 查看后端日志了解详细错误

### Q: 引导问题没有显示？
A: 检查：
1. `question.md` 文件是否存在
2. 文件格式是否正确
3. 查看后端日志

### Q: 进度没有更新？
A: 检查：
1. 会话是否正常创建
2. API调用是否成功
3. 查看浏览器控制台错误

### Q: 导出功能不可用？
A: 检查：
1. PDF导出需要安装 `reportlab`
2. 确保有足够的磁盘空间
3. 检查文件权限

## 🔐 认证问题

### Q: 登录后提示Token无效？
A: 可能原因：
1. Token已过期（30天有效期）
2. Token格式错误
3. 重新登录获取新Token

### Q: 注册失败，提示邮箱已存在？
A: 该邮箱已被注册，请：
1. 使用其他邮箱
2. 或使用手机号注册
3. 或尝试登录

## 📱 使用问题

### Q: 如何重置密码？
A: 当前版本不支持密码重置，请联系管理员。

### Q: 可以删除账户吗？
A: 当前版本不支持账户删除，请联系管理员。

### Q: 数据会丢失吗？
A: 数据存储在数据库中，只要数据库文件不删除，数据不会丢失。建议定期备份。

## 🚀 部署问题

### Q: Docker容器启动失败？
A: 检查：
1. Docker和Docker Compose是否安装
2. 环境变量是否正确配置
3. 端口是否被占用
4. 查看容器日志：`docker-compose logs`

### Q: 生产环境性能问题？
A: 建议：
1. 使用PostgreSQL替代SQLite
2. 启用Redis缓存
3. 使用Nginx反向代理
4. 考虑使用完整架构模式
